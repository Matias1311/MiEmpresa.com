    Sitios web a base de plantillas (CMS)

    Un programador definitivamente no va a programar desde cero, usaría un CMS. Todo depende de los gustos y de la empresa.

    Es recomendable buscar y usar estas plantillas o CMS. Hoy en día hay plantillas para desarrollar páginas y plataformas virtuales. Es una Buena solución para tener algo más rápido sin necesidad de un programador.

    El programador, con conocimientos mínimos sobre los CMS, puede hacer un sitio web en lo que le tarde en buscar la plantilla (20 minutos), que es mucho más rápido que lo que tomaba hacer antes (4 meses).

    El gran trabajo no está en instalar estos CMS, sino en tener la información. Por ejemplo, un colegio puede hacer una plataforma virtual en un día, pero le va a tomar mucho más tiempo generar la información y el contenido.

    Se puede ahorrar el tiempo en pensar el diseño del sitio, pues el CMS propone una arquitectura, colores, letras, y más. Es mucho más fácil corregirlos en estas plataformas, por lo que está bien como borrador para las siguientes fases.

    Ahorra tiempo y presupuesto, porque un proyecto web incluye mucho personal, una sola persona no puede hacerlo todo.

    Aun así, siempre es mejor tener un trabajo más específico y especializado, pues una página web más profesional y única se hace con ingenieros de sistemas y diseñadores gráficos.

    Plataformas CMS

    WordPress es muy flexible, da muchas opciones, y es una de las más usadas. Un programador puede dejar la página lista con los accesos necesarios para solo realizar cambios básicos. Otro especialista puede continuar con el trabajo. Es más ligero que otros CMS.

    Wix puede ser para empezar, como para blogs, que son más flexibles y dinámicos. Está adaptado a muchísimos proyectos por lo que puede ser un poco pesado.

    Joomla permite realizar sitios web dinámicos a través de un panel de administración flexible y sencillo.

    Drupal es multipropósito y configurable, permite añadir otros servicios a un sitio web como foros y blogs.

    Otras Plataformas

    Elementos visuales

    Freepik ofrece una gama de elementos gráficos y a veces ideas para hacer sitios web. Puede complementar el sitio web fácilmente.

    Información

    Google Académico es una versión de Google que está orientado a la búsqueda de información científica y académica.

    Cursos

    Hay muchas fuentes confiables que permiten capacitarse o especializarse. Algunas son Platzi, Coursera, y Youtube.

    Sitios web de herramientas online

    Ofrece información como códigos, sugerencias, consejos para desarrollar, entre otros. Se debe buscar el sitio web de la herramienta que se está usando, como PHP o Java para programación en esas plataformas.

    Hoy en día es mucho más sencillo desarrollar debido al internet y al intercambio de conocimientos. Siempre se debe tener una activa búsqueda de información.
